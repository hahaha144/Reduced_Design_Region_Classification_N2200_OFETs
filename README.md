# Reduced_Design_Region_Classification_N2200_OFETs
This repository contains the process-property dataset for N2200 based OFET devices along with the jupyter notebook code for applying the reduced design region classification algorithm to the dataset.

Contents :

1)dataset.csv is a dataset containing process-property information of N2200 based OFET devices curated from literature sources

2)Reduced_Design_Region_Classification_Algorithm.ipnyb is a jupyter notebook that contains the code to apply the the custom classification algorithm to the OFET dataset
